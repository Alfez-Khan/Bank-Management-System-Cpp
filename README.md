# Bank Management System (C++)

This is a console-based bank management system developed using C++.  
The project focuses on implementing core C++ concepts such as object-oriented programming and file handling.

## Features
- Create bank accounts
- Deposit and withdraw money
- View account details
- Persistent data storage using files

## Technologies Used
- C++
- File Handling (fstream)
- Object-Oriented Programming

## How to Run
1. Compile the program:
   g++ main.cpp -o bank
2. Run the executable:
   ./bank

## Purpose
This project was built to strengthen fundamentals of C++ programming, including classes, data encapsulation, and program logic.
